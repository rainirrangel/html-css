Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> (não semântica).

Nessa frase temos um termo em destaque usando a tag <strong> (semântica).

Itálico / Ênfase

Nessa frase temos um termo em negrito usando a tag <i> (não semântica).

Nessa frase temos um termo em ênfase usando a tag <em> (semântica).

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>.

E no outro parágrafo temos outro texto marcado no final.

Para definir uma alteração apenas nesse elemento utilizamos o parâmetro style="background-color: color;" na tag <mark> específica.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo. com as tags <big> e <small> respectivamente, reforçando que a tag <big> está obsoleta, a tag <small> permanece funcional.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido par dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o texto sublinhado com a tag <u> (não semântica).

Texto sobrescrito

Utilizado para inserir coisas como x20+3, utilizamos a tag <sup>.

Texto subescrito

Utilizado para inserir coisas como H2O, utilizammos a tag <sub>.
